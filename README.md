# CedaSpy Bot v1.0.0

### Creator: Froddo

CedaSpy Bot is a tool that collects player data from Cedapug and Steam using web scraping techniques. It retrieves player stats, matchmaking ratings (MMR), and ban information. This bot operates without the explicit consent of Cedapug or Steam but is designed to respect their platforms and guidelines.

## Features

- Get player stats by Steam ID or player name from Cedapug.
- Calculate the Matchmaking Rating (MMR) of players.
- Check if players have any bans on Cedapug.

## Disclaimer

This bot is for educational and personal use only. It uses web scraping to collect data, which is done without explicit consent from Cedapug or Steam. The bot does not intend to violate the policies or terms of service of either Cedapug or Steam. Please use it responsibly and respect their platforms and guidelines.

# CedaSpy Bot v1.0.0

### Creador: Froddo

CedaSpy Bot es una herramienta que recopila datos de jugadores de Cedapug y Steam utilizando técnicas de web scraping. - Calcular la calificación de emparejamiento (MMR) de los jugadores.
