# CedaSpy Bot v1.0.0

CedaSpy Bot es una herramienta que recopila datos de jugadores de Cedapug y Steam utilizando técnicas de web scraping. Obtiene estadísticas de jugadores, calificaciones de emparejamiento (MMR) e información sobre baneos. Este bot opera sin el consentimiento explícito de Cedapug ni Steam, pero está diseñado para respetar sus plataformas y directrices.

## Características

- Obtener estadísticas de jugadores por Steam ID o nombre de jugador desde Cedapug.
- Calcular la calificación de emparejamiento (MMR) de los jugadores.
- Verificar si los jugadores tienen baneos en Cedapug.

## Descargo de Responsabilidad

Este bot es solo para fines educativos y personales. Utiliza web scraping para recopilar datos, lo que se hace sin el consentimiento explícito de Cedapug ni Steam. El bot no tiene la intención de violar las políticas o términos de servicio de Cedapug ni de Steam. Úsalo de manera responsable y respeta las plataformas y pautas de estas.
